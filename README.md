#  Social Media Impact on Youth

##  Project Overview

This project is a simple HTML-based research website that explores the impact of social media on youth. It presents both positive and negative effects, along with examples, statistics, and educational insights.

---

##  Objectives

* To understand how social media influences young people
* To highlight both advantages and disadvantages
* To promote responsible usage of social media

---

##  Topics Covered

* What is Social Media
* Why Youth Use Social Media
* Types of Social Media Platforms
* Global Usage Statistics
* Positive Impacts
* Negative Impacts
* Conclusion & Awareness

---

## Technologies Used

* HTML5
* Basic Web Design

---

##  Features

* Structured research-style layout
* Images and clickable links
* Embedded YouTube videos
* Informative and educational content

---

## Key Highlights

* Explains real-world social media usage trends
* Covers mental health, communication, and education impact
* Includes examples like Facebook, Instagram, WhatsApp, etc.
